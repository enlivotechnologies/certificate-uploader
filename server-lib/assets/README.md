# Certificate background image

Place your Canva-exported certificate design here as:

**`certificate-bg.jpg`**

- Export from Canva as PNG.
- Do not edit this image in code; it is used only as a full-size static background.
- Certificate dimensions (width × height) are set in `server/.env` as `CERTIFICATE_WIDTH` and `CERTIFICATE_HEIGHT` to match this image (default 1200×800).
